Description

ChapterSweet provides a diverse assembly of useful tools for Technology Student Association chapters, making management easier for advisers and officers, and increasing the quality of the TSA experience for members. Users create accounts associated with a chapter, and are then granted access to ChapterSweet's plethora of functionality. It is web-based, live 24-7, allowing users to access conference rules quickly from their phones, or sick members to participate in real-time event selection from home. Advisers still retain complete control of chapter functions through ChapterSweet; they can restrict page access, promote/demote officers, choose officer permissions from the settings page, delete and edit data, and view all member activity in an easily-accessible activity log on their dashboard.


Pages and Features

Dashboard

  • Adviser Activity Log
  • Account Details
  • Countdown Timer to Next Conference
  • Listing of User's Events
  • Announcements from the Chapter
  • Create Tasks
  • View and Complete Tasks from your Teams
  • Delete Tasks
  • Drop Events
  • View Fundraising Balance

My Chapter

  • View Summary of User Demographics
  • Table of All Chapter Members
  • View Member Emails, Events, Eventpoints, Grades, Ranks, and Balances
  • Advisers can remove users from Events
  • Advisers can Promote / Demote Users
  • View Members' Tasks and Event Progress
  • View and Complete Tasks from your Teams
  • Assign Eventpoints by Groups or Individually
  • Advisers can delete user accounts
  • Create and Delete Obligations
  • Track and Change Users' Status for Obligations

Files

  • Upload Files
  • Edit File Viewability
  • View and Download Files
  • See the Uploader and Date on Files
  • Make Files Private / Public

Event Rules

  • Advisers can upload Event Rules Files
  • View and Download Files
  • Easy Access to Event Rules from Mobile Devices

Minutes

  • The Secretary can upload Meeting Minutes
  • View and Download Minutes
  • Save Time Reading Minutes in Meetings
  • Make Minutes Accessible to the Whole Chapter

Announcements

  • Officers and Advisers can Write Announcements
  • Option to Automatically send as Email to whole Chapter
  • Tracks the Date and Sender
  • Appear on Members' Dashboard
  • Streamline Chapter Communication

Treasurer

  • Listing of Transactions with Essential Information
  • Transact Funds to and from both chapter and member accounts
  • Track Descriptions of Transactions to assist with organization
  • Automatically generate a full ledger of income and expenses

Parliamentarian

  • Access State and National TSA Chapter Team Guides
  • Quick Links to Helpful Chapter Team Study Guides
  • Automatically Generate Randomized Parliamentary Procedure Tests by Difficulty and Number of Questions
  • Take Benchmark Tests with Recorded Scores

Event Selection

  • View All Events, Teams, Members
  • Easily View Minimum and Maximum Members for Events
  • Signup for Events in Realtime with Event Points
  • View Team ID Numbers
  • View Whether an Event is a Qualifier
  • Print the Event Selection Roster Automatically
  • Send an Automated Event Confirmation Email to all Members
  • Advisers can add Qualifier Teams

Adviser Settings

  • Change your Specialized Chapter Code
  • Set Officer Permissions for Posting Files / Announcements, Email Sending, and Eventpoint Assignment

Space Game


This game - without a real title yet - is my first multiplayer venture. Downloading the client executable will allow you to join any IP currently running a server.

Back

Description

Enter a large multiplayer galaxy with nothing but a small ship and your name. Shoot asteroids to harvest resources which can be sold at plants to gain credits, or become a trader and ship resources from planet to planet for a profit. Upgrade your ship at certain planets for large sums of credits and go destroy other players with your superior weaponry, or purchase a cargo freighter and become an expert trader.

Encounter space obstacles like black holes which spell doom, a smuggler's outpost willing to pay for rare goods, and satellites which can heal anyone nearby. With an constantly-expanding universe there's always more to explore.


Instructions

W and S : Move forward and backwards

A and D : Rotate your ship

Spacebar : Shoot

Left-Click Planets : Open planet resource Gui

Right-Click Planets : Open planet government Gui (WIP)

E : Hyperspace jump

Esc : Display properties


Setup

The game is lacking in some user functionality, but if you wish to play it now first download a client and a server executable below. Run the server executable on your computer and leave it running until you are finished playing. Run the client and connect to the IP 127.0.0.1 to connect to the server running on your own machine. This will allow you to play singleplayer.

I will continue to develop the multiplayer setup to be more user-friendly.

If attempting to run the game on a school network, follow these instructions to run the application.

  • Download and unzip the applications if you have not done so already
  • Open Notepad
  • Type " start /d "path" JoshSpaceGameClient03.exe "
  • Replace path with the location of the application, for example C:/Users/JohnDoe/Downloads
  • Save the notepad document as startgame.bat
  • Double click and run startgame.bat

Golf Anywhere

This game was made as a collaboration with B.Cuthill. It's my first game made in Unity, and is still under development.


Description

The player controls a golf ball and club in a scene filled with obstacles one might encounter in everyday life. The goal for the player is shoot the ball into the hole. To achieve this goal, the player must repeatedly shoot the ball around the room, with varying directions and power, and avoiding or using obstacles to their advantage. Colliding with an obstacle will employ a dynamic physics engine, affecting the path of the ball and usually moving the object to change the layout of the room.


Instructions

  • Left Mouse Button : Shoot Ball / Select Menu Item
  • A / D : Rotate Shot Direction
  • W / S : Change Shot Power
  • Q / E : Rotate Camera
  • Spacebar : Reset Ball
  • R : Reset Level
  • L : Next Level (if set in Options Menu)
  • O : Options Menu
  • Escape : Main Menu

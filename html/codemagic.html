CodeMagic

This game was the submission of team 1001-901 to the Pennsylvania Technology Student Association State Conference.


Description

Play as a wizard in a world that revolves around basic coding logic to function. Create spells to interact with a diverse environment and in the process gain a basic understanding of programming logic. Also, you can freeze deer. What more did you need?

Installation

Windows

  • Download the Font File Attached Below
  • Place the file in Control Panel > Personalization and Appearance > Fonts
  • You will need Administrator Rights to do this
  • Open any application like MS Word or Notepad and select the Shaq font to begin
  • Enjoy the man, myth, and legend anywhere you want to type!


Note

In MS Word you may have to disable the option to replace straight quotes with smart quotes.

This is found under Options > Proofing > AutoCorrect Options

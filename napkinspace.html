Description

Napkinspace provides an online canvas, allowing for realtime drawing collaboration with other users. It includes functionality for freehand drawing, text, color and size changing, and both an 'undo' and 'save' feature. Napkinspace takes a simplistic approach to providing users with the invaluable ability to brainstorm, diagram, and visualize together. This project was developed for the 2017-18 Technology Student Association Software Development competition in collaboration with two other individuals.

Process

Generally, when designing a website, I begin with requirement analysis - what does the site have to do? What features must it include? After deciding on the functionality, I find an example site (either one I have already created or one from the world wide web) and use that site as a model for the atmosphere I'm creating with the new website. I diagram a rough layout, determining the positioning and structure of the site's elements and decide on a setup for the navbar. I then import the Bootstrap library, along with JQuery and Popper.js, and any Google fonts I might want to use. From there I begin writing the custom HTML for the site, supported with a custom CSS file (Cascading Style Script). As the site takes shape, I also write JavaScript for any complicated functions, and use JQuery for Bootstrap modals, popovers, and tooltips. If a site requires a form or any form of data collection, I use PHP for the site instead, and occasionally use AJAX to submit data. Once the site is complete, I will present it to the client, and make any changes they require.
